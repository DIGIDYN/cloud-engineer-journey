# HPC & AI Infrastructure Engineer Journey 🚀

Personal learning path to become an expert in High Performance Computing (HPC) and AI Infrastructure.

## 🎯 Main Objectives
- Build a solid foundation in Linux Administration & Scripting.
- Master HPC Cluster Management (SLURM, InfiniBand, MPI).
- Deploy and optimize AI workloads (GPU, CUDA, MLOps).
- Achieve professional certifications: LFCS & CKA.

## 🗺️ Roadmap
- **Phase 0 — Linux Foundations**: Shell, Permissions, Networking (Week 1-4)
- **Phase 1 — Scripting**: Advanced Bash & Python (Week 5-8)
- **Phase 2 — Advanced Linux**: System Tuning & LFCS Prep (Week 9-12)
- **Phase 3 — HPC Core**: SLURM Job Scheduler (Week 13-17)
- **Phase 4 — Distributed Storage**: Ceph & NFS HA (Week 18-22)
- **Phase 5 — GPU & CUDA**: NVIDIA Infrastructure (Week 23-27)
- **Phase 6 — HPC Networking**: InfiniBand & RDMA (Week 28-31)
- **Phase 7 — Containers & K8s**: Docker & GPU-enabled Kubernetes (Week 32-35)
- **Phase 8 — MLOps Infra**: AI Tooling & CKA Prep (Week 36-39)
- **Phase 9 — Portfolio**: Final Project & Certifications (Week 40-43)

## 🛡️ Target Certifications
- **LFCS** (Linux Foundation Certified System Administrator)
- **CKA** (Certified Kubernetes Administrator)

## 🛠️ Stack
**HPC:** SLURM, MPI, InfiniBand, Ceph, Lustre  
**AI:** NVIDIA GPU, CUDA, PyTorch, MLOps  
**Infra:** Linux, Kubernetes, Ansible, Terraform, AWS (GPU instances)  
**Dev:** Python, Bash, Git
