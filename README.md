# Cloud Engineer Journey

Personal learning path to become a Cloud Engineer specialized in Cloud Security and AI Cloud.

## Roadmap
- Phase 0 — Foundations: Linux, Bash, Git, Python, Networking (weeks 1-4)
- Phase 1 — AWS Core + IAM Advanced (weeks 5-10)
- Phase 2 — Terraform IaC + Security (weeks 11-15)
- Phase 3 — Ansible Advanced (weeks 16-18)
- Phase 4 — Docker, Kubernetes, Helm, GitOps (weeks 19-24)
- Phase 5 — CI/CD + DevSecOps (weeks 25-28)
- Phase 6 — Cloud Security + Vault + Prowler (weeks 29-33)
- Phase 7 — Observability: Prometheus + Grafana (weeks 34-36)
- Phase 8 — AI & Cloud: Bedrock, RAG, MLOps (weeks 37-41)
- Phase 9 — Cloud Resume Challenge + GCP + FinOps (weeks 42-45)
- Phase 10 — 5 Certifications (weeks 42-46)

## Certifications Target
- AWS Cloud Practitioner (CLF-C02)
- AWS SysOps Administrator Associate (SOA-C02)
- AWS Solutions Architect Associate (SAA-C03)
- HashiCorp Terraform Associate (003)
- AWS Certified Security Specialty (SCS-C02)

## Stack
AWS · GCP · Terraform · Ansible · Docker · Kubernetes · GitHub Actions · Vault · Prometheus · Grafana · Bedrock
